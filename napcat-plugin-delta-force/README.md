# 🎯 三角洲行动

三角洲行动游戏数据查询插件，支持战绩查询、账号绑定、服务器状态等功能。

## ✨ 功能特性

- 📊 **战绩查询** - 查询玩家 SOL/MP 模式详细数据
- 💰 **金币查询** - 查看角色金币余额
- 🔗 **账号绑定** - 绑定游戏 Token 快速查询
- 🏥 **服务器状态** - 查看游戏服务器健康状态
- 📋 **战局流水** - 查询最近战局记录
- 🎙️ **TTS 语音** - 文字转语音功能，支持多种预设音色
- 🖼️ **图片渲染** - 战绩数据渲染为精美图片
- 🌐 **WebUI 配置** - 可视化管理配置和账号

## 📖 指令说明

| 指令 | 说明 |
|------|------|
| `帮助` / `菜单` | 显示帮助信息 |
| `绑定 <Token>` | 绑定游戏账号 |
| `解绑` | 解除绑定 |
| `查询` / `我的数据` | 查询绑定账号战绩 |
| `金币` | 查询金币余额 |
| `流水 [页数]` | 查询战局流水 |
| `服务器状态` | 查看服务器健康状态 |
| `说 <内容>` | TTS 语音合成 |
| `音色列表` | 查看可用音色 |
