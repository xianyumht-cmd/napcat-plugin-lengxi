# 🎮 meme 娱乐插件

meme 表情包制作、点歌、AI 绘画、哈基米等娱乐功能合集。

## ✨ 功能特性

- 🖼️ **meme 表情包** - 支持多种 meme 模板，自动更新表情数据
- 🎵 **点歌功能** - 搜索并发送音乐卡片，支持 LRU 缓存
- 🎨 **AI 绘画** - 支持预设 Prompt 和自定义绘图
- 🐱 **哈基米制作** - 趣味哈基米内容生成
- 👑 **主人系统** - 添加/删除/查看主人列表

## 📖 指令说明

| 指令 | 说明 |
|------|------|
| `菜单` | 显示功能菜单 |
| `meme <模板> <文字>` | 制作 meme 表情包 |
| `点歌 <歌名>` | 搜索并发送音乐 |
| `画 <描述>` | AI 绘画 |
| `添加主人 <QQ>` | 添加主人（主人专属） |
| `删除主人 <QQ>` | 删除主人（主人专属） |
| `主人列表` | 查看主人列表 |
