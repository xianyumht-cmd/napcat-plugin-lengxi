# ⚡ NapCat 工作流

AI 可视化工作流插件，通过拖拽节点创建自动化流程与定时任务。

## ✨ 功能特性

- 🔗 **可视化编排** - WebUI 拖拽式工作流编辑器
- 🎯 **消息触发** - 支持精确匹配、模糊匹配、正则匹配
- ⏰ **定时任务** - Cron 表达式定时执行工作流
- 📦 **变量存储** - 工作流级别和全局级别的持久化存储
- 🔀 **条件分支** - 根据消息内容、用户、群等条件分流
- 📤 **多种动作** - 发送消息、调用 API、HTTP 请求、CQ 码解析
- 🔒 **权限控制** - 支持主人专属工作流

## 📖 使用方式

1. 打开 WebUI 控制台进入工作流编辑器
2. 拖拽节点创建触发器 → 条件 → 动作的流程
3. 保存并启用工作流

### 节点类型

| 类型 | 说明 |
|------|------|
| 触发器 | 消息匹配（精确/模糊/正则）、定时触发 |
| 条件 | 判断用户、群、消息内容、变量等 |
| 动作 | 发送消息、调用 API、存储变量、HTTP 请求 |
| 存储 | 读写工作流变量和全局变量 |
