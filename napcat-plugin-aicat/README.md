# 🐱 AI Cat - 智能猫娘助手插件

智能猫娘群管助手，支持 AI 对话、群管理、自定义指令等功能。

## ✨ 功能特性

- 🤖 **AI 智能对话** - 基于 GPT 的智能对话，支持多轮上下文
- 👑 **主人管理系统** - 安全的验证码认证机制
- ⚙️ **自定义指令** - 创建自定义触发词和响应
- ⏰ **定时任务** - 支持 AI 定时发送消息和 API 调用
- 👀 **用户检测器** - 监控特定用户并自动执行操作
- 🔧 **群管理功能** - 禁言、踢人、设置管理员等
- 📝 **消息记录** - 群聊/私聊消息日志管理
- 🎨 **AI 绘画** - 支持预设和自定义 Prompt 绘图

## 📖 指令说明

默认指令前缀为 `xy`，可在配置中修改。

| 指令 | 说明 |
|------|------|
| `xy <内容>` | 与 AI 对话 |
| `xy 帮助` | 显示帮助信息 |
| `xy 上下文` | 查看当前对话状态 |
| `xy 清除上下文` | 清除对话历史 |
| `xy 设置主人` | 申请成为主人（验证码输出到日志） |
| `xy 验证主人 <验证码>` | 完成主人认证 |

### 主人专属指令

| 指令 | 说明 |
|------|------|
| `xy 主人列表` | 查看所有主人 |
| `xy 模型列表` | 查看可用 AI 模型 |
| `xy 切换模型 <数字>` | 切换 AI 模型 |
| `xy 检测器列表` | 查看用户检测器 |

### Packet 指令

| 指令 | 说明 |
|------|------|
| `xy api <OneBot API>` | 直接调用 OneBot API |
| `xy pb <hex>` | 发送 Protobuf 数据包 |
| `xy pbl <hex>` | 发送长 Protobuf 数据包 |

## 🛠️ AI 工具能力

AI 可以调用 OneBot API 完成群管理任务：

- 发送消息（群聊/私聊/合并转发）
- 群管理（禁言、踢人、设置管理员、群名片等）
- 获取信息（用户、群、消息等）
- 网络搜索、网页获取
- 自定义指令/定时任务/用户检测器管理
