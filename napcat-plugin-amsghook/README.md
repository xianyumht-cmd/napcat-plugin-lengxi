# 📨 消息代发转卡

让所有插件与 OneBot 的消息通过官方机器人 Markdown 发送，也可以修改其他插件预发送的内容。

## ✨ 功能特性

- 📝 **Markdown 转发** - 将普通消息转换为官方 Markdown 卡片发送
- 🔀 **消息拦截** - Hook 其他插件的消息发送，统一处理
- 🤖 **QQ 官方机器人** - 对接 QQ 官方机器人 API（WebSocket）
- 🖼️ **媒体支持** - 支持图片、语音等媒体消息的转发
- ⚙️ **规则配置** - 按插件名配置不同的转发规则
- 🌐 **WebUI 面板** - 可视化配置管理

## 📖 使用方式

1. 在 WebUI 中配置 QQ 官方机器人的 AppID 和 Secret
2. 设置需要拦截的插件规则
3. 插件会自动 Hook 消息发送流程，将消息转为 Markdown 卡片
