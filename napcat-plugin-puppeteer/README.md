# 🖼️ Puppeteer 渲染服务

NapCat Puppeteer 渲染服务插件，提供 HTML/模板截图渲染 API，供其他插件调用。

## ✨ 功能特性

- 📸 **截图渲染** - 将 HTML 内容渲染为图片
- 🌐 **网页截图** - 对指定 URL 进行截图
- 🎨 **模板渲染** - 支持自定义 HTML 模板 + 数据渲染
- ⚙️ **浏览器管理** - 自动检测/下载 Chrome，支持自定义路径
- 🔄 **健康检查** - 自动检测浏览器状态，断线自动重连
- 🖥️ **WebUI 管理** - 可视化配置浏览器参数和渲染选项
- 🔌 **API 接口** - 提供 RESTful API 供其他插件调用

## 📖 使用方式

本插件为服务型插件，不直接响应用户消息，而是为其他插件提供渲染能力。

### API 接口

| 接口 | 说明 |
|------|------|
| `POST /render/html` | 渲染 HTML 字符串为图片 |
| `POST /render/url` | 对 URL 截图 |
| `POST /render/template` | 模板 + 数据渲染 |

### 依赖说明

需要系统安装 Chrome/Chromium 浏览器，插件支持自动下载。
