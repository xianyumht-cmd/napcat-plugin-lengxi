# 🧧 自动抢红包

自动抢红包插件，监听红包消息自动领取，支持口令红包、黑白名单、随机延迟、防检测等功能。

## ✨ 功能特性

- 🎯 **自动领取** - 监听群红包消息自动抢红包
- 💬 **口令红包** - 自动识别并发送口令
- ⏱️ **随机延迟** - 可配置延迟范围，模拟真人操作
- 🛡️ **防检测** - 抢完后暂停该群一段时间
- 📋 **黑白名单** - 群/用户级别的过滤控制
- ⏰ **时间段禁用** - 指定时间段内不抢红包
- 🔔 **通知推送** - 抢到红包后通知主人
- 💬 **感谢消息** - 抢完自动发送随机感谢语
- 📊 **统计数据** - 累计抢红包次数统计
- 🔄 **热重载** - 配置文件修改后自动生效

## 📖 指令说明

所有指令仅主人可用，前缀为 `#抢红包` 或 `#红包`。

| 指令 | 说明 |
|------|------|
| `#抢红包` | 查看状态和帮助 |
| `#抢红包 开启` / `关闭` | 总开关 |
| `#抢红包 仅通知 开启/关闭` | 仅通知不抢 |
| `#抢红包 延迟 <最小> <最大>` | 设置随机延迟(ms) |
| `#抢红包 延迟 关闭` | 关闭随机延迟 |
| `#抢红包 防检测 开启/关闭 [分钟]` | 防检测开关 |
| `#抢红包 过滤 无/白名单/黑名单` | 切换过滤模式 |
| `#抢红包 黑名单 群/用户 添加/删除 <ID>` | 管理黑名单 |
| `#抢红包 白名单 群/用户 添加/删除 <ID>` | 管理白名单 |
| `#抢红包 时间 开启/关闭 [开始] [结束]` | 时间段禁用 |
| `#抢红包 感谢 添加/删除/列表 [消息]` | 管理感谢消息 |
| `#抢红包 主人 <QQ>` | 设置主人QQ |
| `#抢红包 重置统计` | 重置统计数据 |
