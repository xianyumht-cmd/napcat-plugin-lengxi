# 🔄 插件自动更新

NapCat 插件自动更新管理器，定时检查已安装插件的最新版本并自动更新或提醒。

## ✨ 功能特性

- 🔍 **自动检查** - 定时扫描商店索引，检查已安装插件的新版本
- 📦 **一键更新** - 支持单个或批量更新插件
- 🔔 **通知推送** - 发现更新后推送到指定群/私聊
- 🤖 **自动更新** - 可配置自动更新模式，无需手动操作
- 🌐 **镜像加速** - 内置多个 GitHub 镜像，自动选择最快节点
- 🛡️ **配置保护** - 更新时自动备份并恢复用户配置
- 📥 **GitHub 安装** - 直接从 GitHub 仓库安装插件
- 🖥️ **WebUI 面板** - 可视化管理更新配置

## 📖 指令说明

默认指令前缀为 `更新`。

| 指令 | 说明 |
|------|------|
| `更新 帮助` | 显示帮助信息 |
| `更新 检查` | 检查所有插件更新 |
| `更新 列表` | 查看已安装插件 |
| `更新 状态` | 查看更新状态和配置 |
| `更新 全部` | 更新所有可更新插件（主人） |
| `更新 <编号/插件名>` | 更新指定插件（主人） |
| `更新 模式 <auto/notify>` | 切换更新模式（主人） |
| `更新 间隔 <分钟>` | 设置检查间隔（主人） |
| `更新 忽略 <插件名>` | 忽略指定插件（主人） |
| `更新 取消忽略 <插件名>` | 取消忽略（主人） |
