# 🛡️ 群管插件

全功能群管理插件，集成入群验证、违禁词过滤、防撤回、刷屏检测、问答系统等。

## ✨ 功能特性

- ✅ **入群验证** - 新成员数学验证，超时或答错自动踢出
- 🚫 **违禁词过滤** - 多级惩罚（撤回/禁言/踢出/拉黑）
- 🔄 **防撤回** - 撤回消息自动转发，支持图片/语音等
- 💬 **刷屏检测** - 自动禁言刷屏用户
- ❓ **问答系统** - 精确/模糊/正则匹配的自动回复
- 📛 **黑白名单** - 全局黑名单 + 群独立黑名单
- 🏷️ **名片锁定** - 锁定群成员名片防止修改
- 😀 **回应表情** - 自动回应消息表情
- 📊 **活跃统计** - 群成员发言活跃度排行
- 🔒 **入群拒绝词** - 申请信息含关键词自动拒绝

## 📖 指令说明

### 基础管理（管理员）

| 指令 | 说明 |
|------|------|
| `群管帮助` | 显示完整帮助菜单 |
| `踢出 @某人` | 踢出群成员 |
| `禁言 @某人 [分钟]` | 禁言（默认10分钟） |
| `解禁 @某人` | 解除禁言 |
| `全体禁言` / `全体解禁` | 全体禁言开关 |
| `授予头衔 @某人 内容` | 设置专属头衔 |
| `清除头衔 @某人` | 清除专属头衔 |
| `锁定名片 @某人` | 锁定当前名片 |
| `解锁名片 @某人` | 解锁名片 |

### 针对用户（管理员）

| 指令 | 说明 |
|------|------|
| `针对 @某人` | 自动撤回该用户消息 |
| `取消针对 @某人` | 取消针对 |
| `针对列表` | 查看针对列表 |
| `清除针对` | 清除所有针对 |

### 黑白名单

| 指令 | 说明 |
|------|------|
| `拉黑 @某人` | 加入全局黑名单（主人） |
| `取消拉黑 @某人` | 移出黑名单（主人） |
| `群拉黑 @某人` | 加入本群黑名单（管理员） |
| `白名单 @某人` | 加入白名单（主人） |
| `黑名单列表` / `群黑名单列表` / `白名单列表` | 查看列表 |

### 违禁词 & 问答（主人/管理员）

| 指令 | 说明 |
|------|------|
| `添加违禁词 <词语>` | 添加违禁词 |
| `删除违禁词 <词语>` | 删除违禁词 |
| `违禁词列表` | 查看违禁词 |
| `添加问答 关键词\|回复` | 添加精确问答 |
| `添加模糊问答 关键词\|回复` | 添加模糊问答 |
| `添加正则问答 正则\|回复` | 添加正则问答 |
| `删除问答 <关键词>` | 删除问答 |
| `问答列表` | 查看问答列表 |

### 其他功能

| 指令 | 说明 |
|------|------|
| `开启防撤回` / `关闭防撤回` | 防撤回开关 |
| `开启回应表情` / `关闭回应表情` | 表情回应开关 |
| `添加拒绝词 <词语>` | 入群申请拒绝关键词 |
| `活跃统计` | 查看群活跃排行 |
