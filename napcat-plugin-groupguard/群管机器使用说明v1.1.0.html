总览

本手册已覆盖全部管理员/成员可触发功能，以及所有自动执行功能（事件触发、默认策略触发、辅助模块触发），如需售后请联系2320006072。

✅ 每条指令都提供“格式 + 作用 + 简述 + 复制按钮”。
✅ 自动功能单独列出，避免遗漏。
⚠️ 指令是否生效取决于本群配置与权限。

入群验证与自动审批

开启入群验证 管理员

格式：开启入群验证 / 开启功能 入群验证
作用：新人进群后进入数学题验证流程。
开启入群验证

关闭入群验证 管理员

格式：关闭入群验证 / 关闭功能 入群验证
作用：关闭新人答题验证。
关闭入群验证

自动审批开关 管理员

格式：开启自动审批 / 关闭自动审批（或 开启功能/关闭功能 自动同意）
作用：自动处理入群申请。
开启自动审批

入群暗号策略 管理员

格式：设置入群暗号 内容 / 关闭入群暗号 / 开启暗号回落 / 关闭暗号回落
作用：暗号审核与回落自动审批策略。
设置入群暗号 我已阅读群规

欢迎新人

设置欢迎词 管理员

格式：设置欢迎词 内容
作用：新人入群后发送欢迎消息；支持变量 {user}、{group}。
设置欢迎词 欢迎 {user} 加入 {group}

防撤回与反刷屏

防撤回管理 管理员

格式：开启防撤回 / 关闭防撤回 / 防撤回列表
作用：恢复成员撤回内容并触发通知逻辑。
开启防撤回

反刷屏参数 管理员

格式：开启刷屏检测 / 设置刷屏窗口 秒 / 设置刷屏阈值 次 / 设置刷屏禁言 分钟
作用：频率刷屏处罚。
设置刷屏阈值 10

复读检测 管理员

格式：设置复读阈值 次数
作用：连续重复相同文本触发处罚。
设置复读阈值 4

针对用户自动撤回 管理员

格式：针对 @某人 / 取消针对 @某人 / 针对列表 / 清除针对
作用：针对列表成员消息自动撤回。
针对 @某人

二维码拦截与消息过滤

二维码拦截快捷开关 管理员

格式：开启二维码撤回 / 关闭二维码撤回
作用：识别图片二维码并自动撤回（需要配合“屏蔽二维码”指令使用，否则无效）。
开启二维码撤回

消息类型过滤 管理员

格式：屏蔽 类型 / 取消屏蔽 类型
类型：图片、视频、语音、链接、二维码、名片、小程序、转发。作用：屏蔽后，识别到对应类型会自动撤回
屏蔽 链接

违禁词过滤

违禁词管理 管理员

格式：添加违禁词 词语 / 删除违禁词 词语 / 违禁词列表
作用：命中后自动执行处罚链。
添加违禁词 赌博代理

处罚策略 管理员

格式：设置违禁词惩罚 1~4 / 设置违禁词禁言 分钟
作用：设置撤回、禁言、踢出、拉黑级别。
设置违禁词惩罚 2

自动问答

查看问答 管理员

格式：问答列表
作用：查看当前生效问答规则。
问答列表

格式：精确问XX答YY / 模糊问XX答YY
作用：配置精确与包含匹配回复。

作用：移除指定问答。
删除问答 帮助

自动触发功能总表（无指令也会执行）

以下能力由消息链、事件链或辅助模块自动触发，已完整列出：

自动功能触发条件行为
入群申请自动审核收到 group add request按黑名单→防刷→暗号→拒绝词→回落自动同意决策
新人入群验证会话group_increase 且开启验证发送数学题、超时踢出、答错计次、非数字答案自动撤回
欢迎消息group_increase 且关闭验证自动发送欢迎词
黑名单拦截黑名单成员发言自动撤回消息
白名单豁免白名单成员发言跳过黑名单/刷屏/违禁词等处罚链
问答自动回复消息命中问答规则自动回复（精确/模糊/正则）
针对自动撤回成员在针对列表消息自动撤回
违禁词处罚链命中违禁词撤回，并按等级禁言/踢出/拉黑
消息类型过滤命中过滤规则自动撤回（图/视频/语音/链接/二维码等）
刷屏/复读检测超过频率阈值或复读阈值自动禁言并提示
防撤回恢复收到撤回事件且防撤回开启恢复消息并向主人发送撤回通知
名片锁恢复名片变更事件或消息检测到异常自动还原锁定名片
退群拉黑group_decrease.leave 且开关开启自动加入黑名单
邀请统计与奖励邀请入群成功更新邀请计数并可发放邀请积分
活跃统计记录群消息流转自动累计消息数与今日活跃数据
机器人自身消息自动撤回配置默认开启60秒撤回机器人发出的消息延时撤回
