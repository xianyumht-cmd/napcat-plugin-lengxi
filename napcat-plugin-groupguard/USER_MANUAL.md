# 🛡️ 群管机器人 - 用户使用手册

欢迎租用本群管机器人！本文档将指导您（群主/管理员）如何配置和使用机器人的各项功能。

> **注意**: 部分高级功能（如⭐标记）需要联系老板购买**授权**后方可使用。

## 1. 基础管理 (所有群可用)

无需复杂配置，直接在群内发送指令即可。

| 功能 | 指令格式 | 说明 |
| :--- | :--- | :--- |
| **踢人** | `踢出 @某人` | 将成员移出群聊 |
| **禁言** | `禁言 @某人 <分钟>` | 默认10分钟，如 `禁言 @张三 60` |
| **解禁** | `解禁 @某人` | 解除禁言 |
| **全体禁言** | `全体禁言` / `全体解禁` | 开启/关闭全员禁言 |
| **警告系统** | `警告 @某人` | 记录一次警告 |
| | `查看警告 @某人` | 查看当前警告次数 |
| | `清除警告 @某人` | 清空警告记录 |
| **设置警告** | `设置警告上限 <次数>` | 达到次数自动惩罚（默认3次） |
| | `设置警告惩罚 <禁言/踢出>` | 达到上限后的动作（默认禁言） |

## 2. 进阶群管 (⭐ 需授权)

### 2.1 入群与安全
*   **入群验证**:
    *   `开启入群验证` / `关闭入群验证`: 新成员进群需回答数学题，超时或答错自动踢出。
    *   `设置欢迎词 <内容>`: 设置进群欢迎语，支持 `{user}`(AT用户) 和 `{group}`(群号)。
    *   `开启自动审批` / `关闭自动审批`: 自动通过无异常的入群申请。
*   **退群管理**:
    *   `开启退群拉黑` / `关闭退群拉黑`: 成员退群后自动加入本群黑名单，禁止再次进入。

### 2.2 消息净化
*   **二维码拦截**: `开启二维码撤回` / `关闭二维码撤回` (防止发广告引流)
*   **媒体过滤**:
    *   `屏蔽图片` / `取消屏蔽图片`
    *   `屏蔽链接` / `取消屏蔽链接`
    *   `屏蔽视频` / `取消屏蔽视频`
    *   `屏蔽小程序` / `取消屏蔽小程序` (防止恶意卡片)
*   **刷屏检测**: `开启刷屏检测` / `关闭刷屏检测` (短时间大量发消息自动禁言)
*   **防撤回**: `开启防撤回` / `关闭防撤回` (有人撤回消息时，机器人自动重新发送出来)

### 2.3 针对模式
*   **针对**: `针对 @某人` (该人发送的所有消息都会被秒撤回)
*   **取消**: `取消针对 @某人`
*   **列表**: `针对列表`

### 2.4 宵禁 (夜间模式)
*   **开启**: `开启宵禁 <开始时间> <结束时间>` (如 `开启宵禁 00:00 06:00`)
*   **关闭**: `关闭宵禁`

### 2.5 定时任务
*   **添加**: `定时任务 <时间> <内容>` (如 `定时任务 08:00 大家早上好！`)
*   **删除**: `删除定时任务 <ID>`
*   **列表**: `定时列表`

## 3. 互动运营 (增加群活跃)

### 3.1 积分系统
*   **签到**: 发送 `签到` (每日一次，获得积分，连续签到有奖励)。
*   **排行**: 发送 `签到榜` 查看今日排行。
*   **查询**: 发送 `我的积分`。
*   **抽奖**: 发送 `抽奖` (消耗积分抽取大奖)。

### 3.2 邀请裂变
*   **查询**: 发送 `邀请查询` (查看自己拉了多少人)。
*   **排行**: 发送 `邀请榜`。
*   *机制*: 邀请新成员进群，邀请人可获得积分奖励（需开启）。

### 3.3 发言奖励
*   **开启**: `开启发言奖励 <积分数>` (如 `开启发言奖励 1`，每说一句话+1分)
*   **关闭**: `关闭发言奖励`

### 3.4 智能问答
*   **添加**: `添加问答 关键词|回复内容` (如 `添加问答 官网|访问 www.example.com`)
*   **模糊匹配**: `添加模糊问答 关键词|回复` (包含关键词即触发)
*   **列表**: `问答列表`
*   **删除**: `删除问答 <关键词>`

## 4. 其他指令

*   `群管帮助`: 获取简易指令菜单。
*   `运行状态`: 查看机器人运行时间、版本和统计信息。
*   `授权查询`: 查看当前群的授权到期时间。
*   `锁定名片 @某人`: 防止某人修改群名片（改了会自动改回来）。
*   `解锁名片 @某人`: 解除锁定。
